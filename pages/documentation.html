SnailDB: A Lightweight Python Database

SnailDB is a lightweight, non-SQL database for Python, designed to be simple and easy to use. Developed with simplicity and efficiency in mind, SnailDB provides basic CRUD (Create, Read, Update, Delete) operations along with query functionalities. BETA VERSION

Why SnailDB?

Lightweight and Simple

Unlike traditional SQL databases, SnailDB doesn't require complex setups or installations. It's designed to be lightweight, making it perfect for small to medium-sized projects where simplicity is key.

Easy Integration

With a simple API, integrating SnailDB into your Python projects is straightforward. Whether you're a beginner or an experienced developer, SnailDB's intuitive interface makes data management hassle-free.

Persistent Storage

Data is stored in a JSON file, ensuring persistence across sessions. This means you don't have to worry about losing your data when the application is restarted.

Basic CRUD Operations

SnailDB supports basic CRUD operations, allowing you to create, read, update, and delete data with ease. Whether you're storing user information, product details, or any other data, SnailDB has you covered.

Query Functionality

Need to retrieve specific data? SnailDB offers query functionality that allows you to filter data based on field name, operator, and target value. This feature enhances data retrieval efficiency, saving you time and effort.

Quick Start SnailDB

                
                
                from snaildb import SnailDB
                # Create a SnailDB instance
                db = SnailDB("database.json", "data")
                # Insert data
                db.insert({"key": "1", "name": "John", "age": 30})
                # Get data
                print("Key '1' data:", db.get("1"))

Examples SnailDB

                
                
                # Setup the database on db
                db = SnailDB("my_database.json", "data")
                # Example 1: Insert a document using insert() function
                db.insert({"key": "1", "name": "Akhil", "age": 20})
                # Example 2: Insert multiple documents using insert_all() function
                db.insert_all({"key": "3", "name": "Jon", "age": 10}, {"key": "4", "name": "Ali", "age": 21})
                  
            
                
                # Example 3: Update the database
                # For updating the database, use the key value.
                db.update("1", {"name": "John", "age": 31})
    
                # Example 4: View the database
                # Print data without pretty formatting:
                print(db.get("1"))
    
                # Print data with pretty formatting:
                print(db.get("1", pretty=True))
                  
            
                
                # Example 5: View all the database
                # Print all data without pretty formatting:
                print(db.get_all())
    
                # Print all data with pretty formatting:
                print(db.get_all(pretty=True))

Delete SnailDB

                
                
                # Example 6: Delete data
                # For deleting data, enter the key
                db.delete("2")
    
                # Example 7: Delete the entire database
                # The db database will be deleted
                db.drop()

Query SnailDB

                
                
                # Query Function without pretty formatting
                test = db.query("age", "<", 30)
                print(test)
                # Query Function with pretty formatting
                test = db.query("age", "<", 30, pretty=True)
                print(test)
                # We can use all operators
                #Greater Operator
                test = db.query("age", ">", 30, pretty=True)
                print(test)
                #Lesser Operator
                test = db.query("age", "<", 30, pretty=True)
                print(test)
                #Equal to operator
                test = db.query("name", "=", "Ali", pretty=True)
                print(test)
                # Lesser and Equal to operator
                test = db.query("age", "<=", 29, pretty=True)
                print(test)
                # Greater and Equal to operator
                test = db.query("age", ">=", 29, pretty=True)
                print(test)
                # Not Equal to operator
                test = db.query("age", "!=", 29, pretty=True)
                print(test)
                # Query Method with Skip Method
                test = db.query("age", "<", 30, skip=1, pretty=True)
                print(test)
                # Query Method with Limt method
                test = db.query("age", "<", 30, limt=1, pretty=True)
                print(test)
                # Query Method with Limt method and skip method
                test = db.query("age", "<", 30, skip=1, limt=1, pretty=True)
                print(test)
                print("Key '1' data:", db.get("1"))
